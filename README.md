## About The Project

Reusable boiler plate of API for an project made with MongoDB, Express & Typescript

### Built With

* [Typescript](https://www.typescriptlang.org/)
* [MongoDB](https://www.mongodb.com/atlas/database)
* [Express](https://expressjs.com/)
